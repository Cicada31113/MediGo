# 03_03 · 데모 스크립트 – MediGo

> 문제 3 · 시연 시 따라 읽을 스크립트

## 1. 환경
- 사용 빌드: `mediGo-desktop.exe`
- 데이터: 기본 시드 약국(시청/광화문 등)

## 2. 단계별 플로우

1) **잠금 화면에서 시작**
   - 멘트: “지금 보시는 화면은 사용자 휴대폰 잠금 화면입니다.”
   - 클릭/스와이프 → 해제 → 알림 배너 탭

2) **앱 진입 및 처방전 확인**
   - 멘트: “알림을 통해 방금 발급된 처방전을 바로 확인할 수 있습니다.”

3) **수령 지역/약국 선택**
   - 검색 → 서울시청 인근 약국 리스트 → 하나 선택  
   - 멘트: “복잡하게 약국을 찾지 않아도, 현재 위치 기준으로 배달 가능한 약국만 모아 보여줍니다.”

4) **주문 생성 및 배달 현황**
   - 멘트: “이제 약국에서 배송지까지 실제 경로를 따라가는 라이더를 보실 수 있습니다.”  
   - ETA와 상태 뱃지(배정 완료 → 배달 중 → 배송 완료) 강조

5) **배송 완료**
   - 완료 화면 + 토스트 메시지 강조  
   - 멘트: “이제 약은 집 앞에 도착했고, 환자는 병원에서 바로 집으로 돌아와 쉴 수 있습니다.”

## 3. Fallback 전략
- API 오류 시: 토스트에 “오프라인 모드” 문구 안내
- exe 문제 시: 미리 녹화된 시연 영상 재생
