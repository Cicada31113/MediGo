# 03_06 · 릴리즈 노트 v1 – MediGo

> 문제 3 · 현재 텀프로젝트 버전의 스냅샷

## 1. 버전 정보
- 버전: v1.0.0 (텀프로젝트 제출용)
- 날짜: 2025-11-24

## 2. 포함 기능
- 잠금/홈 화면 UX + 알림 진입
- 약국 검색/리스트/지도 표시
- 처방전 썸네일 및 업로드 API(로컬 저장)
- 주문 생성/배달 경로 애니메이션/ETA 표시
- 로컬 SQLite 기반 상태 저장
- 무설치 데스크톱 exe 패키지

## 3. 알려진 이슈
- 실제 약국/병원 시스템과는 아직 연동되지 않음
- 네트워크 없는 환경에서는 일부 기능이 제한될 수 있습니다

## 4. 롤백/플래그 전략

  - exe 재시작 또는 녹화 영상으로 전환
- 향후 기능 플래그:
