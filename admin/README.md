# 메디-고 관리자 대시보드

## 주요 기능

- 📋 주문 관리 (접수, 조제 중, 배달 중, 완료)
- 💊 복약 지도 작성 및 발송
- 📸 약봉투 사진 업로드 (AI 학습 데이터 수집)
- 📊 통계 및 분석 (추후 구현)

## 시작하기

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 주요 페이지

- `/` - 대시보드 홈
- `/orders` - 주문 목록
- `/orders/:id` - 주문 상세
- `/training-data` - AI 학습 데이터 관리
